# Інструкція для редакторів сайту (Decap CMS /admin)

Ця інструкція — для власників та редакторів сайту **Volunteer Gang**. Вона
пояснює, як самостійно змінювати тексти, збори та учасниць через адмінку —
**без програмування**.

Адмінка: **https://www.vgang.com.ua/admin**

---

## 1. Що це таке

Сайтом можна керувати через зручну панель (адмінку). Там ви редагуєте контент у
готових формах — заголовки, суми збору, фото учасниць тощо. Усі поля мають
українські підписи, тож орієнтуватися легко.

Важливо розуміти структуру:

- Є **чотири розділи** (у меню зліва). Кожен відповідає за свою частину сайту.
- Ви можете **редагувати** наявні записи, але **не можете** додавати чи видаляти
  самі розділи. (Виняток — список учасниць усередині «Головної»: там записи
  можна додавати й видаляти.)

---

## 2. Як увійти

1. Відкрийте **https://www.vgang.com.ua/admin**
2. Натисніть кнопку входу (**Login / Login with Netlify Identity**).
3. Введіть свою **пошту та пароль**.

**Перший вхід:** ви отримаєте лист-запрошення на пошту. Перейдіть за посиланням
із листа й задайте пароль.

**Забули пароль:** натисніть **«Forgot password»** на вікні входу — прийде лист
для відновлення.

Якщо запрошення не приходить або не вдається зайти — зверніться до розробника,
щоб він додав вашу пошту в **Netlify → Identity**.

---

## 3. Важливо перед початком

Прочитайте ці три правила — вони економлять нерви:

1. **Збереження = публікація.** Після натискання **Publish** зміни йдуть на
   живий сайт (з’являються за кілька хвилин, поки сайт перезбирається). Окремого
   «чернеткового» режиму немає.
2. **Майже всі поля обов’язкові.** Не залишайте їх порожніми — інакше сайт не
   збереже зміни. Порожніми можна лишати тільки **фото** та **Instagram**
   учасниці.
3. **Посилання вводьте повністю** — з `https://`. Посилання на банку — це лінк
   з Monobank (наприклад `https://send.monobank.ua/jar/...`).

## 4. Де що редагувати

Нижче — усі чотири розділи меню й що саме вони змінюють на сайті.

### 🛠 Налаштування сайту → Загальні

Загальні елементи, що є **на кожній сторінці**: шапка, меню, підвал, назва
бренду.

| Поле | Що змінює |
|---|---|
| **Бренд → Слово 1 / Слово 2 (графіті)** | Логотип-напис (наприклад `VOLUNTEER` / `GANG`) |
| **Слоган** | Гасло бренду |
| **Діємо з (рік)** / **Регіон** | Рік заснування та регіон |
| **Слоган у футері** | Напис у підвалі сайту |
| **Кнопка (CTA) → Текст / Посилання** | Кнопка «Докинути в банку» в шапці |
| **Меню** | Пункти навігації у шапці (текст + посилання). Можна додавати/видаляти пункти |
| **SEO → Title / Description** | Заголовок і опис сайту для Google та соцмереж |

> ⚠️ Кнопка «Докинути в банку» в шапці показується, лише коли увімкнено активний
> збір (див. розділ **Активний збір** нижче).

### 💛 Активний збір

Керує блоком **«Активний збір»** на головній сторінці — картка збору з ціллю та
кнопкою на банку.

| Поле | Що змінює |
|---|---|
| **Показувати секцію** | Головний перемикач. **Вимкнено → зникає і картка збору на головній, і кнопка «Докинути в банку» в шапці.** Увімкніть, коли збір активний |
| **Збір №** | Номер збору (напр. `47`) |
| **Статус (бейдж)** | Напис на бейджі (напр. `АКТИВНИЙ`) |
| **Заголовок картки** | Назва збору |
| **Опис картки** | Короткий опис під заголовком |
| **Текст праворуч** | Абзац поруч зі статистикою |
| **Ціль, ₴** | Сума цілі в гривнях (тільки число, без пробілів і знаків) |
| **Лінк на банку** | Посилання на банку Monobank |
| **Текст кнопки** | Напис на кнопці (напр. `Докинути в банку →`) |

### 🖼 Дані для шаблонів

Заповнює **початкові значення** для генератора Instagram-картинок на сторінці
**`/templates`**. Самі картинки ви створюєте й завантажуєте вже на сторінці
`/templates` — тут лише зручні значення «за замовчуванням».

| Поле | Що змінює |
|---|---|
| **День збору** | Номер дня, який показується на картинках |
| **Заголовок для шаблонів** | Головний заголовок (кожен рядок — з нового рядка) |
| **Акцентний рядок (бригада/підрозділ)** | Виділений кольором рядок заголовка |
| **Опис для шаблонів** | Короткий опис на картинках |
| **Ціль, ₴** / **Зібрано, ₴** | Використовуються для смужки прогресу на картинках |
| **Лінк на банку** | Посилання на банку |

### 📄 Сторінки → Головна

Усі тексти головної сторінки (крім блоку активного збору) **та список
учасниць**.

| Поле | Що змінює |
|---|---|
| **SEO → Title / Description** | Заголовок і опис головної сторінки для пошуку |
| **Hero → Надзаголовок / Бейдж «діємо з» / Вступний текст** | Верхній екран сайту |
| **Заголовки секцій** | Номери й назви трьох секцій: «Активний збір», «Банда», «Шаблони» |
| **Статистика (3)** | Три плитки з цифрами (підпис, значення, колір) |
| **Рядок «докази»** | Рядок на кшталт `ЧЕКИ ✓ ФОТО ПЕРЕДАЧІ ✓` |
| **Учасниці** | Список банди (див. «Типові задачі» → додати учасницю) |
| **Тизер шаблонів → Текст / Кнопка** | Блок-запрошення до генератора шаблонів |

## 5. Типові задачі

### ▶️ Відкрити / закрити активний збір

1. Меню → **Активний збір**.
2. Перемикач **«Показувати секцію»**:
   - **Увімкнено** — на головній видно картку збору, у шапці з’являється кнопка
     «Докинути в банку».
   - **Вимкнено** — і картка, і кнопка зникають.
3. Оновіть **Заголовок картки**, **Ціль, ₴**, **Лінк на банку** під поточний
   збір.
4. Натисніть **Publish**.

### 💰 Змінити ціль або текст збору

Меню → **Активний збір** → змініть **Ціль, ₴** / **Заголовок картки** /
**Опис картки** → **Publish**. Ціль вводьте лише цифрами, наприклад `500000`.

### 👤 Додати або відредагувати учасницю

1. Меню → **Сторінки** → **Головна**.
2. Знайдіть список **«Учасниці»**.
3. Щоб додати — натисніть кнопку **«Add …» / «+»** унизу списку. Щоб змінити —
   розгорніть потрібний запис.
4. Заповніть:
   - **Номер (01…)** — порядковий номер (напр. `05`).
   - **Ім’я** та **Роль**.
   - **Фото** — натисніть поле й завантажте зображення (можна не додавати).
   - **Instagram (хендл без @)** — лише нік, без `@` та без посилання
     (напр. `a.lina_belko`). Можна лишити порожнім.
   - **Колір рамки** — оберіть зі списку (`pink`, `acid`, `bubble`, `chalk`).
5. **Publish**.

> Порядок учасниць на сайті = порядок у списку. Перетягуйте записи, щоб змінити
> послідовність.

### 🧭 Змінити меню або кнопку в шапці

Меню → **Налаштування сайту** → **Меню** (пункти навігації) або **Кнопка (CTA)**
(головна кнопка) → **Publish**.

### 📸 Підготувати дані для Instagram-шаблонів

Меню → **Дані для шаблонів** → заповніть поля → **Publish**. Далі відкрийте
сторінку **`/templates`** на сайті, де вже згенеруються картинки для завантаження.

---

## 6. Як зберегти та опублікувати

1. Внесіть зміни у форму.
2. Натисніть **Publish → Publish now** (згори праворуч).
3. Зачекайте кілька хвилин — сайт перезбереться, і зміни з’являться на
   **https://www.vgang.com.ua**.

Якщо після збереження бачите помилку — найчастіше десь **порожнє обов’язкове
поле**. Заповніть його й збережіть ще раз.

## 7. Коротка шпаргалка

| Що змінити | Де в адмінці |
|---|---|
| Логотип, слоган, підвал | Налаштування сайту |
| Меню в шапці | Налаштування сайту → **Меню** |
| Кнопка «Докинути в банку» (текст/лінк) | Налаштування сайту → **Кнопка (CTA)** |
| Показати/сховати збір + кнопку в шапці | Активний збір → **Показувати секцію** |
| Ціль, назва, банка збору | Активний збір |
| Учасниці банди (фото, ім’я, IG) | Сторінки → Головна → **Учасниці** |
| Тексти головної (hero, статистика) | Сторінки → Головна |
| Дані для Instagram-картинок | Дані для шаблонів |
